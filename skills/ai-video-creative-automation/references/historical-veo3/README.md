# Historical Story Generation (VEO 3)

Use this reference to execute the 6-step prompt pack for cinematic historical automation.

## Step 1: Story Generation
Generate a 12-scene historical story.
- **Rules**: Visual only, YouTube-safe, 9+ historical events, 1 character anchor.
- **Goal**: Establish the narrative arc from inciting incident to resolution.

## Step 2: Consistency Lock
Create the DNA for characters and the world Bible.
- **Components**: Character skin/outfit, World colors/lighting, Camera movement rules.

## Step 3: VEO 3 Prompts
Convert scenes into technical video prompts.
- **Critical**: Every prompt MUST include the Consistency Header from the Lock.

## Step 4: Clean Export
Extract only the prompts in a numbered list (Autoflow ready).

## Step 5: Narration Script
Write a mysterious, cinematic voice-over script matching the visuals.

## Step 6: Voice Style
Define instructions for the AI Voice model (Tone, pacing, drama).
