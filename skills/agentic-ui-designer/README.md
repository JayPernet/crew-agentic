# Agentic UI Designer

This skill provides the theoretical and practical framework for designing user interfaces that interact with autonomous AI agents. It ensures that AI interactions are transparent, controllable, and visually premium.

## UI Ecosystem Pipeline

This skill is the crucial third phase in the advanced UI capabilities pipeline:

1. **`frontend-designer-specialist` (Foundation):** Establishes the core design system, tokens, geometry, and high-end aesthetic baseline (colors, typography).
2. **`hero-designer` (The Hook):** Operates on top of the design system to craft immersive, high-converting "above the fold" experiences that captivate the user.
3. **`agentic-ui-designer` (The Interaction - This Skill):** Steps in when the user engages with AI capabilities. It dictates how the system behaves, communicates its "thoughts", and allows human intervention, all while maintaining the premium standards set by the foundational skills.

## Usage

Activate this skill when designing chat interfaces, copilot sidebars, agent reasoning logs, dynamic data generators, or any UI where the system makes autonomous decisions on behalf of the user. Ensure you have the `design-system.html` built by the frontend-designer-specialist before starting.
