Paleta de Cores - Hierarquia Cromática

Abaixo está a justificativa semântica de cada cor escolhida:

Background
[PREENCHER HEX]
Cor dominante (60-70% da interface). Base neutra para todo conteúdo.
Surface
[PREENCHER HEX]
Cards e containers elevados. Contraste sutil com background.
Primary Action
[PREENCHER HEX]
CTAs principais, links, estados ativos. Contraste 4.5:1 com background.
Secondary Action
[PREENCHER HEX]
Ações secundárias, highlights, elementos decorativos.
Accent
[PREENCHER HEX]
Destaques, badges, alertas importantes, call-to-attention.
Text Primary
[PREENCHER HEX]
Texto principal, headings. Contraste máximo para legibilidade.
Text Secondary
[PREENCHER HEX]
Texto secundário, labels, metadados. Contraste WCAG AA.
Border
[PREENCHER HEX]
Bordas, divisores, separadores visuais sutis.
